Создай полную архитектуру и прототип веб-сервиса, который объединяет сайт клана Discord и собственный видеохостинг, с полной интеграцией с Discord.

Функционал должен включать:
    1.    Сайт клана с регистрацией пользователей через Discord, модификациями, базой данных, системой логики и ролями.
    2.    Видеоплатформу, где пользователи могут:
    •    Загружать длинные видео;
    •    Генерировать ссылку на видео;
    •    Отправлять ссылку в Discord, чтобы отображалось превью видео, которое можно смотреть прямо в чате;
    •    Система лайков, просмотров, комментариев и статистики.
    3.    Интеграцию с Discord через бота и/или вебхуки, чтобы автоматически отправлять уведомления о новых видео и обновлениях.

Требования к архитектуре:
    •    Серверная часть (бэкенд) с API для сайта, видеоплатформы и Discord.
    •    Хранение видео с использованием облачного хранилища (например, S3/Google Cloud).
    •    База данных для пользователей, видео, лайков, просмотров и комментариев (SQL или NoSQL).
    •    Фронтенд с загрузкой видео, плеером, системой лайков и превью для Discord.
    •    Генерация Open Graph тегов для красивого превью ссылок.
    •    Продуманная система модерации и ролей.

Вывод должен включать:
    •    Схему архитектуры (сервер, БД, фронтенд, API, Discord);
    •    Примеры структуры базы данных;
    •    Пример API-запросов для загрузки видео, получения статистики и интеграции с Discord;
    •    Пример HTML-кода превью для Discord-ссылки;
    •    Рекомендации по хостингу и масштабированию.

Сделай промт максимально понятным для разработчика или ИИ, чтобы он мог полностью создать рабочий прототип такой системы.
