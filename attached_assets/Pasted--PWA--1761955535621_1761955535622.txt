Создай современную веб-панель (PWA-приложение) для игрового клана с подключением к серверу, базой данных, интеграцией Discord и функциями искусственного интеллекта. Приложение должно иметь красивый футуристичный дизайн с эффектами стекла, тумана, металлических бликов и плавных анимаций. Главная страница содержит логотип и эмблему клана, анимированный 3D или живой фон, последние новости и события, кнопки “Войти через Discord”, “Вступить в клан” и “Панель игрока”. После входа через Discord OAuth2 пользователь попадает в свой профиль, где отображаются его ник, аватар, ID, ранг, очки активности, достижения, внутренняя валюта (например, Wave или кристаллы), история событий и доступ к клановому магазину. В панели клана должны быть таблица участников с фильтрацией по активности, графики вклада и роста, календарь рейдов и турниров, панель администратора для управления ранками, валютой и новостями, а также интеграция с API игры (если возможно) для автоматического обновления данных. Магазин позволяет покупать предметы за внутриигровую валюту с анимацией (свечение, вспышка, вращение) и обновлением баланса в реальном времени.

Добавь AI-персонажа — “Советника клана” в виде чата или 3D-аватара, который общается с пользователями, анализирует статистику и даёт советы (“твой вклад выше среднего на 12%”), создаёт новости, поздравления и тексты хроник клана. Сделай возможной AI-озвучку (Text-to-Speech) и генерацию художественных записей (“в этот день клан завоевал новую территорию”). Сайт должен быть связан с Discord через OAuth2 и Webhooks: вход по Discord-аккаунту, публикация новостей из сайта в Discord-каналы, синхронизация рангов и активности.

Используй стек: Frontend — HTML, CSS, JavaScript (или React при необходимости); Backend — Node.js (Express); база данных — MongoDB или Firebase. Сайт должен быть PWA — устанавливаться на устройства как приложение, работать офлайн и иметь адаптивный дизайн. Обеспечь плавные переходы, эффект стеклянных панелей, туманные и неоновые акценты, живой интерфейс, визуализацию данных и интеграцию всех компонентов в единую экосистему клана с искусственным интеллектом, Discord-подключением, магазином, профилями и серверной базой данных.
